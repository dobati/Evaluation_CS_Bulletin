ARTIKEL & SATZALIGNIERUNG


1. MAKE A DIRECTORY MIT PARALLELEN XML DATEIEN

2. SCHREIB EIN SCRIPT, DAS FÜR JEDE DATEI DIE <article> TAGS extrahiert und eine neue XML 
Datei schreibt, in welcher die <article> Tags mit den gleichen "n" Attribut gespeichert 
werden. 

Input:

<book lang="de">
  <article n="0">
    <div> DE </div>
    <div> DE </div>
  </article>
  <article n="1">
    <div heading="article">DE TITEL 1</div>
    <div> DE </div>
  </article>
  <article n="2">
    <div heading="article"> DE TITEL 2</div>
    <div> DE </div>
    <div> DE </div>
  </article>
</book>
  
  
Output:

<?xml version='1.0' encoding='UTF-8'?>
<TEI>
<teiHeader>
JahrgangX_AusgabeY
</teiHeader>
<linkGrp targType="yearbook" xtargets="JahrgangX_AusgabeY_de.xml;JahrgangX_AusgabeY_fr.xml" lang="de;fr">
	<linkGrp targType="article" xtargets="a1;a1">
	</linkGrp>
</linkGrp>
</TEI>


		for each Jahrgang_Titel:
			Bulletin = Jahrgang_Titel[:9]
			Jahr = Jahrgang_Titel[9:13]
			Ausgabe = Jahrgang_Titel[14]
			Name = Jahrgang_Titel[16: Jahrgang_Titel.index(‘_’)]
			Language = Jahrgang_Titel[-6: Jahrgang_Titel.index(‘.’)]

			Store in dict
			If Jahrgang & Ausgabe == gleich:
				make new filename with Jahrgang & Ausgabe + “-Aligned” & store in Aligned_XML
			if not gleich:
				ake new filename with Jahrgang & Ausgabe + “-NoParallel” & store in No_Parallel_XML
